## AL-MR-GP

Active Learning based Multi-fidelity Residual Gaussian Process modelling for data-efficient geomaterial property prediction.

This framework combines abundant low-fidelity data with sparse site-specific high-fidelity data. Active learning is used to select informative high-fidelity observations for calibrating the multi-fidelity residual model.
